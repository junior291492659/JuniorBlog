# Junior_Lee 个人博客介绍

本博客基于 React(负责前后台界面)和 Egg.js(负责服务接口)进行开发，旨在学习 React hooks 和 Typescript.

[博客线上地址](http://129.204.231.203:82/)

本项目为博客前后台的前端项目，后台服务项目暂不公开，但是后文会对其项目结构进行介绍。
